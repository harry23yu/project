Welcome to the PDF AI Summarization Tool!

This tool allows you to summarize PDF files with text in it. Simply use the "upload
PDF file" button to choose a PDF file that you want to summarize, then click the
"Summarize!" button. A summarization of the text should then appear below. You can
use the tool as many times as you want, so if you refresh the page, the summary should
disappear and you can upload another PDF file.
